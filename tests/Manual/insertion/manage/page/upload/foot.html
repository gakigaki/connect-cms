【CSVファイル項目説明】

項目名 必須 説明
page_name 必須 ページ名
permanent_link 必須 固定リンク
background_color 必須 背景色
・HTMLで指定できる色キーワード（例：red, blue）やRGB色（例：#000000, #111）等を設定できます。
・空の場合、NULL と入力します。
header_color 必須 ヘッダーバーの背景色
・HTMLで指定できる色キーワード（例：red, blue）やRGB色（例：#000000, #111）等を設定できます。
・空の場合、NULL と入力します。
theme 必須 テーマ
・Connect-CMSの public\themes ディレクトリ配下のテーマ名を入力します。（例：Defaults/Blue）
　標準のテーマ名は、githubのpublic\themes ディレクトリ で確認できます。
・設定なしの場合、NULL と入力します。
layout 必須 レイアウト
NULL : 未設定
0|0|0|0 : メインのみ
0|0|0|1 : フッター
0|0|1|0 : 右
0|0|1|1 : 右、フッター
0|1|0|0 : 左
0|1|0|1 : 左、フッター
0|1|1|0 : 左、右
0|1|1|1 : 左、右、フッター
1|0|0|0 : ヘッダー
1|0|0|1 : ヘッダー、フッター
1|0|1|0 : ヘッダー、右
1|0|1|1 : ヘッダー、右、フッター
1|1|0|0 : ヘッダー、左
1|1|0|1 : ヘッダー、左、フッター
1|1|1|0 : ヘッダー、左、右
1|1|1|1 : ヘッダー、左、右、フッター
base_display_flag 必須 メニュー表示
1 : 表示する
0 : 表示しない

※1 インポートすると、「公開設定」は 公開 で登録します。

【CSVファイル例】

"page_name","permanent_link","background_color","header_color","theme","layout","base_display_flag"
"アップロード","/upload","NULL","NULL","NULL","NULL","1"
"アップロード2","/upload/2","NULL","NULL","NULL","NULL","1"

※ 文字コードは Shift_JIS
